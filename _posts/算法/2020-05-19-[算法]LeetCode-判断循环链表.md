## 遍历链表法

> 创建一个容器存放遍历过程中的链表,在每次遍历的时候去判断目前的结点是否已经存在在容器中.如果存在容器中说明已经走了一个循环,那么就是循环链表

```java
public class Solution {
    private Set cache = new HashSet();

    public boolean hasCycle(ListNode head) {
        while (head != null) {
            if (cache.contains(head)) {
                return true;
            }
            cache.add(head);
            head = head.next;
        }
        return false;
    }
}
```
